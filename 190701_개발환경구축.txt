1. 자바 개발 환경
	1-1. 자바 언어를 개발 및 실행 프로그램
		http://java.sun.com
		v1.8
		
		-> 설치 -> path 설정
		
		: 주의사항
			jdk의 bin path는 oraclee db path 보다 우선이어야 함
	
	
		: 질문 1 : jdk가 제대로 설치되었는지 확인했던 명령어?
			:cmd 창을 반드시 새로 실행해야 함
			  새롭게 설정한 path 값은 새로 실행해야만 인식
			>java -version
			
		- 질문 2 : cmd창 실행 단축키
			윈도우버튼 + r
		
		
		
		
	1-2. 개발 tool
		eclipse 다운로드
			: javaEE 버젼
			http://www.eclipse.org
		압축 해제
			: 문제소지 최소화
				: 경로를 짧게 이클립스 이동
				1.dev/eclipse/eclipse.exe
				- eclipse.exe를 작업표시줄에 빠른 아이콘 등록
		주의사항
			: 반드시 jdk가 우선 path 설정이 되어 있어야 함


2. eclipse 작업 단계
	1단계 : project 생성
	2단계 : 소스 개발(*.java)
	3단계 : 컴파일
		- 정상 컴파일 후 *.class 자동 생성
		- 이클립스 동작 : 정상 저장이 자동 컴파일
	4단계 : 실행
		- *.class로만 실행
	5단계 : 실행 결과 보고 확인
